# KoSh Bot Telegram

TElegram Bot for SMM Voting Services with balance management, custom services, and admin panel.

## Features
- Vote Orders (A, B, C, D, E)
- Custom Services (C1, C2)
- User Balance Management
- Admin Panel
- Easypaisa/JazzCash Payment Support
- Order Status Tracking

## Deploy
```bash
npm install
node index.js
```
